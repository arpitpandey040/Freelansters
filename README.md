## Installation ðŸ’¾

```bash
git clone https://github.com/Bhanu1776/Freelansters.git
```

Create config.env file in `backend` folder and Fill your `.env` variables:

```env
PORT=
DATABASE=
SECRET_KEY=
```

Install deps:

```bash
npm install
```

Run React server from `Client` folder :

```bash
npm start
```

Simultaneously Run Node.js Server :

```bash
node server.js
```
